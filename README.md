## Info

This is a administration panel.

Built using TypeScript, React, Redux Toolkit, react-router, sass, [vite](https://vitejs.dev/), [dummyjson](https://dummyjson.com/), Github Actions and [Github Pages](https://pages.github.com/).

## Installation

To run this project locally, you need to follow these steps.

1. Clone the repository using the following command:

```
git clone https://github.com/legerakun/administration-panel
```

2. Navigate to the project directory:

```
cd administration-panel
```

3. Setup dependencies:

```
npm install
```

4. Start local server:

```
npm run dev
```

5. Open your browser and visit local website page.
